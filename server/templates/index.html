🎯 Automated Fine-Tune

This wizard walks through every character position from farthest to nearest, letting you correct modules that land on the wrong flap.

How it works:
1. All modules home first (~12 sec)
2. All modules move to the same character
3. Click modules that show the NEXT character (overshot) → apply correction
4. Click modules that show the PREVIOUS character (undershot) → apply correction
5. Confirm all correct, move to next character

Default 25 steps (~40% of one character spacing). Increase for larger errors.
63 = farthest from home (default). Lower to skip already-tuned chars.
